# Problem Description

<b>The Fibonacci numbers are defined as:</b>

- F(0) = 0
- F(1) = 1
- F(n) = F(n - 1) + F(n - 2) for n > 1

<b>Task: </b>

Given an integer n, return the nth Fibonacci number.

# Approach Used

<b>Instead of using recursion (which can be inefficient due to repeated calculations), this solution uses an iterative approach:</b>

- Start with the first two Fibonacci numbers: 0 and 1
- Iteratively calculate the next Fibonacci number up to n
- Store only the last two values to keep space usage minimal
